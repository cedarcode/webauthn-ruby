# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.y   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you have discovered a security bug, please send an email to security@cedarcode.com
instead of posting to the GitHub issue tracker.

Thank you!
